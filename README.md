# About
The landing page for a KickStarter project.

The project was created with:
- HTML5
- CSS3
- SASS
- JavaScript
- Figma
- BEM

Landing page has the responsive layout for various screen-widths:
  - Large screens 2560px
  - Full HD 1920px
  - The design 1600px
  - Notebook 1280px
  - Tablet 1024
  - Mobile (> 320px)
